# CubismSdkForWeb 安装指南

## 步骤

1. 访问 https://www.live2d.com/sdk/download/web/ 下载 CubismSdkForWeb
2. 解压后将 `Core/live2dcubismcore.min.js` 复制到本目录：
   ```
   frontend/public/live2d/core/live2dcubismcore.min.js
   ```
3. 启动前端开发服务器，Live2D 功能即可正常使用

## 注意事项

- CubismSdkForWeb 受 Live2D 专有软件许可协议约束
- 本项目不内置 SDK 文件，需用户自行下载
- Live2D 模型文件同样需要用户自行准备合规模型
